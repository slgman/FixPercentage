# Support & Troubleshooting

If you are experiencing issues with this mod, have a feature request, or just need some help, you're in the right place.

## Reporting Bugs

The best way to report a bug or unexpected behavior is by opening an issue on the GitHub repository. To help fix the issue as quickly as possible, please include the following in your report:

* **A clear description** of the problem.
* **Steps to reproduce** the issue (e.g., "Place a startpos, reverse trigger, and die").
* Your **Geometry Dash version** and **Geode version**.
* Any relevant **crash logs** (these can be found in your `geode/crashlogs` folder).

[Open an Issue on GitHub](https://github.com/slgman/FixPercentage/issues)
